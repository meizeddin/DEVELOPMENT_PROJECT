Lesson 3: Logical Equivalence and Laws of Propositional Calculi

Logical Equivalence

Logical equivalence is a concept in logic and mathematics that describes a relationship between two logical statements. Two statements are said to be logically equivalent if they have the same truth value in all possible scenarios or situations. In other words, if the truth value of one statement is true, then the truth value of the other statement is also true, and if the truth value of one statement is false, then the truth value of the other statement is also false.

Example:

The statements "P and Q" and "Q and P" are logically equivalent. This is because both statements are true only if both P and Q are true, and false otherwise. Similarly, the statements "not (P or Q)" and "not P and not Q" are also logically equivalent, as both statements are true only if both P and Q are false, and false otherwise.

Importance:

Logical equivalence is an important concept in logic and mathematics, as it allows us to simplify complex logical expressions and proofs by replacing one statement with another that is logically equivalent. By doing so, we can reduce the number of logical steps required to reach a conclusion, making our reasoning more efficient and effective.

Practical Example:

Two propositions are logically equivalent if they have identical truth tables, see below.

Consider the truth tables of
¬ (P & Q) and (¬P | ¬Q)

P Q ¬ (P & Q)
True True False
True False True
False True True
False False True
P Q ¬P ¬Q (¬P | ¬Q)
True True False False False
True False False True True
False True True False True
False False True True True

Since the truth tables are the same, the propositions are logically equivalent, and we can write:
¬(P & Q) ≡ (¬P | ¬Q)

Some Important Laws of Propositional Calculi

1. Idempotent Laws

P | P ≡ P

P & P ≡ P

2. Commutative Laws

P | Q ≡ Q | P

P & Q ≡ Q & P

3. Identity Laws

P | F ≡ P

P & T ≡ P

4. Associative Laws

P | (Q | R) ≡ (P | Q) | R

P & (Q & R) ≡ (P & Q) & R

5. Distributive Laws

P & (Q | R) ≡ (P & Q) | (P & R)

P | (Q & R) ≡ (P | Q) & (P | R)

6. Complement Laws

P & ¬P ≡ F           ¬T ≡ F

P | ¬P ≡ T           ¬F ≡ T

7. De Morgan’s Laws

¬(P & Q) ≡ ¬P | ¬Q

¬(P | Q) ≡ ¬P & ¬Q

8. Double Negation

¬(¬P) ≡ P
