Lesson 2: Propositional Connectives and Truth Tables

Connectives

The connectives are:

Name Symbol Description
Negation ¬/! Not
Conjunction & And
Disjunction | Or
Implication => If...Then
Equivalence <=> If and only if
Unary Operators Binary Operators


A unary operator takes a single operand(P/statement)

A Binary operator takes two operands(P and Q)

Negation

Given any statement p, another statement, called the negation of p, can be formed by writing “It is false....” before p or, if possible, by inserting in p the word “not”.
Symbolically,
¬ p
Denotes the negation of p (read “not p”)

P ¬P
True False
False True

Conjunction

Any two statements can be combined by the word “and” to form a compound statement called the conjunction of the original statements.
Symbolically,
p & q
Denotes the conjunction of the statements p and q, read “p and q”.

P Q P & Q
True True True
True False False
False True False
False False False

Disjunction

Any two statements can be combined by the word “or” (in the sense of “and/or”) to form a new statement which is called the disjunction of the original two statements.
Symbolically,
p | q
Denotes the disjunction of the statements p and q, read “p or q”.

P Q P | Q
True True True
True False True
False True True
False False False

Conditional (Implication)

Many statements, particularly in mathematics, are of the form “If P then Q”. Such statements are called conditional statements and are denoted by P => Q
The conditional P => Q frequently read “P implies Q” or “P only if Q”.

P Q P => Q
True True True
True False False
False True True
False False True

Implication is false only in the case when the antecedent (P) is true but the consequence (Q) is false.
The last two cases are difficult to explain.
Basically if we consider the expression if P then Q where P is false, then it does not matter whether q is true or false. The implication has not failed in these cases, so the resulting value is defined to be true.

P Q ¬P (¬ P) | Q
True True False True
True False False False
False True True True
False False True True

Observe that the above truth tables are identical. Hence, P => Q is logically equivalent to the proposition (¬P) | Q:

P => Q ≡ (¬P) | Q

In other words, the conditional statement “If P then Q” is logically equivalent to the statement “Not P or Q” which only involve the connectives | and ¬ and thus was already a part of our language.

Bi-conditional (Equivalence)

Another common statement is of the form “P if and only if Q”. Such statements are called biconditional statements and are denoted by P <=> Q.

P Q P <=> Q
True True True
True False False
False True False
False False True


<=> - Equivalence is true if the propositions are both true or both false.

Tautology -- Contradiction -- Contingency

If an expression always results in a true value, the proposition is called a tautology.

Such like
P ¬P P | ¬P
True False True
False True True

If an expression always results in a false value, the proposition is called a contradiction.

Such like
P ¬P P & ¬P
True False False
False True False

If an expression results in both true and false values, the proposition is called a contingency.

Such like
P Q P & Q
True True True
True False False
False True False
False False False
